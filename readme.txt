Наборы выбора в базе данных игры.

Игры )).

======================================
Дополнительные функции в окнах хелпера:

Статистика экспедиции гильдии -> Стоимость медалей.
Обзор города -> Планировщики города и квантового поселения. Подсчёт количества зданий.
Обзор производства -> Все колодцы на карте.
Строения рейтинг эфф. -> Статы игрока + Здания игрока.
Поля гильдии -> Отправка данных в Телеграм. Информация о процентах на ближайших секторах.
Инфобокс -> Отправка данных в Телеграм и Дискорд.
Калькулятор вложений -> Стоимость ВС Титана.
